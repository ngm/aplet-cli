# aplet-cli

Command line tools for test-driven agile product line engineering.

See https://aplet.readthedocs.io/ for a bit of background.

## basic usage

    $ mkdir my-awesome-pl
    $ cd my-awesome-pl
    $ aplet init
    $ aplet makedocs
    $ aplet servedocs
    $ vi aplet.yml
    $ aplet runtests
